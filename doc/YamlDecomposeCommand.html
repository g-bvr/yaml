YamlDecomposeCommand

Purpose

Decompose a yaml file into a folder tree

Variants

UsageDescription
DECOMPOSE YAML yaml INTO targetDecompose a yaml file into a folder tree

Arguments

ArgumentValue
yamlThe path to the yaml file (relative to current workspace)
targetThe path of the result folder (relative to current workspace, will be created including ancestors if not present, yet)
